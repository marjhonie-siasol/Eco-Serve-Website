# 🌿 Eco-Serve Website Application (Beginner Friendly Front-End)

Welcome to the **Eco-Serve Website** repository! This project is a simple, static e-commerce website designed for a plant shop. 

This repository is built entirely with **pure HTML, CSS, and Vanilla JavaScript (No Frameworks)**. It is the perfect starting point for beginners who want to explore web development, practice CSS styling, or understand basic JavaScript DOM manipulation.

## 🎯 Purpose of this Project

This repository serves as a learning resource and a foundational playground for new developers. Feel free to clone this project, use it as a base, and improve it on your own! You can practice your skills by:
- Modifying the layout or creating a dark mode theme.
- Adding JavaScript form validation to the login or contact sections.
- Making the shopping cart completely functional using `localStorage`.
- Refining the mobile responsiveness.

## 🚧 Future Developer Roadmap

This is currently just a **raw, static front-end prototype**. However, development will not stop here! 

In the future, I plan to continue developing this project into a fully dynamic, full-stack application by introducing:
- A dynamic **Backend Server** to handle application logic.
- A **Database Integration** to safely store product inventories, user accounts, and real-time orders.
- A functional user authentication system (Logins and Sign-ups).

## 💻 Included Tech Stack

- **HTML5:** Clean, semantic structure and website layout.
- **CSS3:** Custom styling and responsive design techniques.
- **JavaScript (Vanilla):** Basic interactivity and functional elements (like the countdown timer).
- **Swiper.js:** For smooth and interactive image carousels.
- **Font Awesome:** Included for lightweight, scalable vector icons.
